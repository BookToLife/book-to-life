# Book to Life

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This application lets share books between people. You will use it with your friends and coworkers to see their books and arrange a temporally exchange.

### Contribute
This project is still under development. You can contribute to it by reporting a bug, submitting a fix or proposing new features.

Please take a look at the [contribute file](https://github.com/BookToLife/book-to-life/blob/main/contributing.md) to know how to start.

Also, there is a [Kanban board](https://github.com/orgs/BookToLife/projects/1) where you can learn more about the status of the project.
